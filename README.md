## OkHttpTools简介
----------
一个对OkHttp封装的简单易用型HTTP请求和文件下载管理框架
* 简化OkHttp使用
* 支持GET,POST,PUT,DELETE,HEAD,PATCH
* 支持文件下载
* 支持返回bean对象
* 支持返回json String数据
* 支持返回JsonObject对象
* 支持https请求
* 支持文件上传
* 其他
